Welcome to the API Documentation

To build the server­side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

API Endpoints

Business Logic/What does it do? Endpoint URL Query Parameters Method Type Request Data Format Example Request Response Data Format Example Response
Get List of All Movies /movies N/A GET Nothing in body of request N/A A JSON object holding data about all movies See "Get data about single movie by title" for each item format.
Get data about single movie by title /movies /:Title GET Nothing in body of request N/A A JSON object holding data about a single movie, also containing an ID. { id: 1, title: 'Arrival', director: 'Dennis Villeneuve', year: 2016, genre: 'Science Fiction', 'Drama' description: 'A linguist works with the military to communicate with alien lifeforms after twelve mysterious spacecrafts appear around the world.', image URL: 'https://www.imdb.com/title/tt2543164/mediaviewer/rm3938516992/' }
Get data about genre by name/title (Ex: "Thriller") /movies/genres /:Name GET Nothing in body of request N/A A JSON object holding description of genre. { Thrillers: 'This genre of movies are characterized and defined by the moods they elicit, giving viewers heightened feelings of suspense, excitement, surprise, anticipation and anxiety.' }
Get data about a director /movies/directors /:Name GET Nothing in body of request N/A A JSON object holding data about a single director. { id: 2, director: 'Dennis Villeneuve', biography: 'Denis Villeneuve OC CQ is a French Canadian film director, producer, and screenwriter. He is a four-time recipient of the Canadian Screen Award for Best Direction...etc.' birthdate: 'October 3, 1967' }
Allow new users to register /users N/A POST A JSON object holding data about the user to add. { username: 'KSFlynn007', birthday: April 7, 1994 } A JSON object holding data about the user that was added, including a generated ID#. { id: 27, username: 'KSFlynn007', birthday: April 7, 1994 }
Allow users to update their user info(username) /users /:id PUT Nothing in body of request N/A A JSON obejct holding updated data about user. { id: 27, username: 'SciFiFan007', birthday: April 7, 1994 }
Allow users to add a movie to their favorite list /users/:id /:FavoriteMovie POST A JSON object holding data about the movie to add to favorites list. { title: 'Spotlight', director: 'Tom McCarthy', year: 2015, genre: 'Biography', 'Crime', 'Drama' description: 'The true story of how the Boston Globe uncovered the massive scandal of child molestation and cover-up within the local Catholic Archdiocese, shaking the entire Catholic Church to its core.', image URL: 'https://www.imdb.com/title/tt1895587/mediaviewer/rm899739136/' } A text message confirming addition of movie to favorites list. 'The movie 'Spotlight' has been added to your favorites.'
Allow users to remove a movie from their list of favorites /users/:id /:FavoriteMovie DELETE Nothing in body of request N/A A text message confirming removal of movie from favorites list. 'The movie ______ has been removed from your favorites.'
Allow users to deregister /users /:id DELETE Nothing in body of request N/A A text message confirming deregistration of user. 'Thank you, your account has been removed.'
